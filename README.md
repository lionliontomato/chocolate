# 巧克力の歌單

這是依照原本「布丁の歌單」版型修改的靜態網站版本。

## 修改內容

- 網站名稱改為：巧克力の歌單
- 色系改為：淺棕色／巧克力色系
- Google 試算表連結改為：https://docs.google.com/spreadsheets/d/1V9bLbkkXWM1DbMotAEPWewF9j3qfQZkX2OObY1DtCjw/edit?usp=sharing
- 其餘版型、搜尋、分類標籤、隨機抽歌功能維持原本配置

## 上傳方式

把 `index.html`、`style.css`、`script.js` 三個檔案上傳到新的 GitHub repository，
再到 Settings → Pages → Deploy from a branch → main / root，即可產生網址。
